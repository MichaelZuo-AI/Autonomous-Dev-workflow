# Autonomous Mobile Dev

## What This Repo Is

A methodology framework for AI-driven mobile app development. Contains strategy docs, workflow reference, and reusable templates — not app code.

## Repo Structure

- `STRATEGY.md` — Leadership alignment doc (the "why")
- `docs/workflow.md` — Full 5-layer workflow reference (the "how")
- `templates/` — Copy into app repos when starting a project
- `scripts/` — Verification scripts to include in app repos

## Conventions

- Templates use `[placeholder]` syntax for values to fill in
- All spec templates follow Given/When/Then format
- Design tokens are prescriptive — no freestyle styling in app projects
